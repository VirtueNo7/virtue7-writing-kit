# Workspace

This directory is a placeholder for an active user's project. A capable AI system may create or update:

- `input/` - supplied notes, interviews, or source material;
- `canonical/` - kernel, matrix, invariant set;
- `research/` - source and claim records;
- `planning/` - book plan and unit briefs;
- `manuscript/` - authored prose;
- `validation/` - findings and approvals.

Do not place personal or private user material in the public repository.
